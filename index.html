AutoDS v2.0

"A Bloomberg-inspired, local-first AI data intelligence platform — drop in any CSV and get complete analysis, drift detection, ML training, and autonomous AI insights. High performance."

Problem Statement

Imagine you get a new dataset. Before any real work begins, you spend hours just understanding it — writing code to check for missing values, plotting distributions, testing correlations. Then if you train a model and it fails, you debug that manually too. AutoDS eliminates all of that repetitive groundwork in one place.

Literature Review / Market Gap

Tools like Google AutoML require cloud access and billing. Jupyter is powerful but manual. pandas-profiling gives static reports with no AI reasoning. No existing local tool combines autonomous AI agents, data contract enforcement, drift detection, and model failure debugging in one interface.

Research Innovation

AutoDS uses an MCP-style tool server architecture — the AI agent (Qwen2.5-Coder via NVIDIA API) reasons and plans, while a separate Python tool layer safely executes real analysis. This strict separation makes the system auditable, extensible, and production-safe.

System Methodology

Input

Upload any CSV via drag-and-drop or paste an absolute file path. Clean, simple ingestion with no setup required.

Architecture

Four-layer pipeline: React Dashboard UI → FastAPI REST/WebSocket → NVIDIA Qwen2.5 agent → MCP Python tool registry. Each layer has one job and communicates cleanly.

Output

Interactive charts, statistical tables, Trust Score, model comparisons, drift alerts, and auto-generated Pandas cleaning scripts — all live in the dashboard.

System Architecture

UI Layer — React Dashboard

Bloomberg-inspired UI. Dense information layout, zero decorative whitespace. JetBrains Mono + Inter typography. Color-coded output: cyan for data, amber for warnings, green for passing, red for failures.

FastAPI Backend

REST + WebSocket server on port 8000. Handles file uploads, manages analysis state, streams real-time updates to the frontend, and coordinates all MCP tool calls.

NVIDIA AI Agent — Qwen2.5-Coder

Powered by the NVIDIA API platform. Runs a strict autonomous loop — reads dataset context, reasons about what analysis to run, calls specific Python tools like multivariate_pca() or run_anova(), and synthesizes plain-text answers.

MCP Tool Registry

The safe execution layer. Pure Python tools using pandas, scikit-learn, and scipy. Covers profiling, correlation, PCA, VIF, K-Means, model training, T-test failure analysis, contract validation, and drift detection.

Key Features

Advanced Data Profiling

Univariate Analysis

16+ stats per column — mean, std, skewness, kurtosis, and entropy. Understand every column before touching it.

Bivariate Analysis

Auto-selects Pearson/Spearman, ANOVA, Chi-square, or Cramér's V based on variable types — no manual setup.

Multivariate Analysis

PCA, VIF multicollinearity check, K-Means clustering, and feature importance — all in one pass.

Correlation Heatmap

Interactive Pearson matrix with critical variable pairs highlighted for instant insight.

Enterprise Data Governance

Data Contract Validator

Enforce schema rules — value limits, regex patterns, uniqueness. Outputs a dynamic 0–100 Trust Score weighted by violation severity vs. row count.

Dataset Compatibility Analyzer

Jaccard overlap to find viable join keys. Flags format mismatches (leading zeros, type conflicts, case issues) and forecasts row loss for Inner/Left joins.

Drift Detection

KL Divergence-based fingerprinting flags distribution shifts, mean drift, and missing value surges between dataset versions.

Machine Learning Engine

Model Recommendation

Intelligently suggests algorithms based on class imbalance flags, row count, and VIF scores — before you even start training.

Auto Model Training

Trains 3 models in parallel with automated 80/20 splits, median imputation, StandardScaler, and LabelEncoder. Compare results side-by-side instantly.

Model Failure Analysis

Isolates misclassified rows, runs T-tests against correct predictions. Outputs plain-English explanations of exactly where and why the model failed.

Data Intelligence

Auto Cleaning Script

Generates ready-to-run Pandas code tailored to fix the exact anomalies found during analysis. No guesswork, just copy and run.

Data Quality Score

Always-visible holistic wellness grade for your dataset. Know your data's health at a glance before running any analysis.

Autonomous AI Agent

Qwen2.5-Coder via NVIDIA API runs multi-step reasoning loops. Asks questions, runs tools, and synthesizes answers — all without you writing a single line of code.

Challenges Faced

•Building a reliable agentic loop where Qwen2.5 consistently calls the correct tool with the right parameters
•Implementing KL Divergence-based drift detection that works on real-world noisy datasets
•Making the Trust Score formula fair — balancing violation severity, row count, and constraint type
•Keeping the dashboard UI performant while streaming multiple WebSocket updates simultaneously

Key Learnings

•Strict separation of AI reasoning and code execution is essential for safe, auditable AI systems
•Statistical heuristics (Jaccard, KL Divergence, T-tests) are far more reliable than simple threshold checks
•Real-time WebSocket streaming fundamentally changes how users trust and interact with analysis tools
•NVIDIA API gives cloud-quality LLM performance without needing any local GPU infrastructure
